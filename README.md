## 软件简介

1. **多Excel文件批量加载与管理**
   - 支持 `.xls`、`.xlsx` 双格式Excel文件批量导入，自动识别表格表头，适配不规则列名
   - 自动为每条数据补充「表格文件名」固定字段，可精准追溯数据来源
   - 内置重复文件检测机制，自动跳过已加载文件，避免数据冗余

## 快速上手操作指南
### 1. 加载Excel数据文件
1. 启动软件后，点击顶部「选择Excel文件(.xls/.xlsx)」按钮
2. 在弹出的文件选择窗口中，选中需要处理的Excel文件（支持多选），点击「打开」
3. 软件自动完成数据加载与解析，顶部状态栏实时更新已加载文件数、数据条数、识别字段数
4. 若加载失败，可根据弹窗提示安装对应依赖库或检查Excel文件格式

### 2. 数据查询操作
1. 切换至「数据查询」标签页
2. 在「查询字段」下拉框中，选择需要检索的目标字段
3. 在「查询内容」输入框中，输入需要检索的关键词
4. 点击「查询」按钮，即可查看所有匹配的结果，弹窗提示匹配数据条数
5. 点击「清空结果」按钮，可一键清空查询输入框与结果表格

1. 切换至「小工具」标签页，默认进入「多次同班同学识别」功能
2. 确认已加载至少2个Excel文件，点击「开始识别」按钮
3. 等待软件完成分析，识别完成后弹窗提示匹配结果数量
4. 可在结果表格中查看完整匹配信息，支持关键词搜索、结果重置、一键清空

## 兼容性详细说明

| 文件格式 | 兼容情况 | 依赖要求 | 注意事项 |
|----------|----------|----------|----------|

| .xls（Excel 97-2003版本） | 完全兼容 | 需安装xlrd==1.2.0版本 | xlrd 2.0及以上版本已移除.xls格式支持，必须安装1.2.0版本 |

## 常见问题FAQ
### Q1：启动软件/加载文件时提示「未安装openpyxl/xlrd库」怎么办？
A：若使用源码运行方式，打开命令提示符/终端，运行以下命令安装对应依赖即可：
```bash
pip install openpyxl xlrd==1.2.0
```
若使用Windows开箱即用版，出现此提示请重新下载完整打包版本。

### Q2：身份证号显示为科学计数法，无法正常识别怎么办？
A：此问题是Excel中身份证号列被设置为数字格式导致，解决步骤：
1. 打开对应Excel文件，选中身份证号所在的整列
2. 右键点击→选择「设置单元格格式」→切换至「数字」选项卡→选择「文本」→点击确定
3. 重新保存Excel文件，再次加载到软件中即可正常识别

### Q3：多次同班同学识别结果为空？
A：请检查以下前置条件是否满足：
1. 已加载至少2个Excel文件（单文件无法识别「多次」出现的组合）
2. 加载的Excel文件表头包含「姓名」「班级」相关的字段（如「学生姓名」「所在班级」）
3. 数据中确实存在在2个及以上表格中同时出现的二人组合

### Q4：Windows高分辨率屏幕下软件界面模糊怎么办？
A：可通过以下步骤修复：
1. 右键软件exe文件→选择「属性」
2. 切换至「兼容性」选项卡→点击「更改高DPI设置」
3. 勾选「替代高DPI缩放行为」，下拉菜单选择「应用程序」→点击确定
4. 重新启动软件即可正常显示

### Q5：软件会上传我的数据吗？如何保障数据安全？
A：软件全程无联网功能，所有数据处理均在用户本地设备完成，不会上传任何数据至外部服务器。使用完成后，点击「重置所有数据」按钮，即可一键彻底清空所有已加载数据、分析结果，无任何本地数据残留，保障用户数据隐私安全。
